Quickstart

Talk to Golpo in 60 seconds.

Install the SDK, pass an API key, send a message. That is the whole quickstart.

terminal
# 1. Install
$ npm install @golpo/sdk

# 2. Set your key
$ export GOLPO_API_KEY=sk_live_...

# 3. Run
$ golpo talk "How was your day?"
